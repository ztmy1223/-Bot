# 🀄 Mahjong Discord Bot (my_mahjong_bot)

麻雀の対局結果を記録・管理し、レーティング（Rate）やポイント（Points）、各種スタッツを自動で計算して Discord 上にランキングやグラフとして表示する Discord ボットです。DeepL API を使用したメッセージの多言語翻訳機能も搭載しています。

---

## 🚀 主な機能

1. **プレイヤー管理・登録**
   * `/register` コマンドによるプレイヤー登録（名前、学年の設定）。
   * 管理者による名前変更（`!rename`）、学年変更（`!regrade`）、年度替わりの自動進級（`!april`）。
2. **対局結果の自動計算・記録**
   * `!submit` コマンドで4人の素点を入力するだけで、ウマオカを考慮したポイント（Points）や、実力に応じたレーティング（Rate）の変動を自動計算。
   * 成績に応じたDiscordロール（段位）の自動昇格・降格通知。
3. **詳細なスタッツ・グラフ表示**
   * `/stats` コマンドで、現在のRate、Points、対局数、平均順位、順位率（1位〜4位率）、直近の対局日を表示。
   * これまでのRate変動履歴を綺麗にビジュアル化した折れ線グラフ（画像）を自動生成して返信。
   * 全試合の対戦履歴から自動計算された **Nemesis（天敵：一番負けている相手）** と **Prey（カモ：一番勝っている相手）** を表示。
4. **対局募集・卓作成**
   * `/set` コマンドで対局の参加者を募集（✅ ❌ のリアクション付き）。
   * リアクションがついたメンバーから「対局作成」コンテキストメニューを使って、自動でランダムに席（東家〜北家）を決定し、1卓または2卓を自動生成。
5. **多言語翻訳**
   * DeepL API を連携させ、メッセージへのリプライで `!en` (英語), `!jp` (日本語), `!zh` (中国語) と打つだけで翻訳結果を返信。

## 📦 フォルダ構成

本ボットは、メンテナンス性を高めるために役割ごとにファイルを分割した「Cogs構造」を採用しています。

```text
my_mahjong_bot/
├── .env                  # 【非公開】環境変数（トークン・パスワード類）
├── .gitignore            # GitHubに秘密ファイルを上げないための除外設定
├── README.md             # この説明書ファイル
├── main.py               # ボットの起動・Cogs自動読み込みメインスクリプト
├── database.py           # MySQLデータベース接続設定・共通SQLクエリ
├── utils.py              # グラフ描画・ランキング更新などの共通関数・辞書

    ├── register.py       # 登録・ユーザー管理（/register, !rename 等）
    ├── stats.py          # 成績確認・手動ランキング更新（/stats
